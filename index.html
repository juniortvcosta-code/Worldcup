Firebase setup required before playing

Step 1 — Create Firebase Project

  1. Go to console.firebase.google.com
  2. Click "Add project", name it anything (e.g. wc2026)
  3. Disable Analytics → Create project

Step 2 — Enable Email/Password Auth

  1. Authentication → Sign-in method
  2. Enable Email/Password → Save

Step 3 — Create Firestore Database

  1. Firestore Database → Create database
  2. Start in Test mode → pick region → Done

Step 4 — Paste Your Config

  1. Project Settings (⚙) → Your apps → Add web app (</>)
  2. Register, copy the firebaseConfig object
  3. Open firebase-config.js and paste your values
  4. Reload this page
